Demo for robot arm

This page uses Cognitive AI for a demo of a robot arm with a simulation of a production line for filling a wine bottle, capping it, and packing it into a box that takes six bottles. Click the play button to start. You can also click the buttons to show/hide the listings for the facts and rules.

This page uses Cognitive AI for behaviours. See an earlier implementation using promises, and the initial experiments with a swinging robot arm.
